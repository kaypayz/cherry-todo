# Cherry To-Do Lists 🍒🍒

This is a cute and colorful two-list to-do tracker.  
Each list has 20 items, with 🍒🍒 icons next to each task.  

## Features
- Two separate lists
- 20 tasks in each list
- Non-negotiable weekly tasks
- Week-by-week automatic color change
- Cute cherry icons 🍒🍒

## How to use
1. Open the `index.html` file in your browser
2. See your two lists with cherries and colorful backgrounds
3. Complete your non-negotiable tasks weekly!

## Notes
- Colors change automatically each week
- Tasks are written in English
- Perfect for a cute personal to-do tracker!
